# Packs

Packs help teams focus the workflow on relevant domains.

- `backend`: API/service implementation emphasis
- `data`: migration and integrity emphasis
- `frontend`: UI and UX implementation emphasis
- `security`: threat/risk and hardening emphasis
- `performance`: latency/throughput optimization emphasis

Use at install:

`npx ranker-agentic-workflow init . --packs backend,data,security`
